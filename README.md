## Getting started with wasm-pack & npm

Get wasm-pack [here!](https://rustwasm.github.io/wasm-pack/installer/)

For a system with Webassembly, run
```bash
wasm-pack build
```

Go web dir, run
```bash
cd ./web
```

Install package.json file's dependency, run
```bash
npm install
```

And enjoy ğŸ˜€, run
```bash
npm run start
```
